The YoloThreat Platform

We were going to develop a whole platform... but we were lazy. It turns out there are 3 big things new threat intel teams need to be able to do: collect data, analyze malware, monitor maliciousness on the internet, and store it all for later.

To that end YoloThreat has borrowed (mostly) & created (only when absolutely necessary) a number of Docker containers for common threat intelligence tools:

  • Maltrieve
  • Combine
  • Scumblr
  • CRITs
  • MISP
  • ELK
  • Viper
  • Thug
